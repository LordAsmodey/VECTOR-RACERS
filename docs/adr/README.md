# Architecture Decision Records (ADR)

ADR фиксируют значимые технические решения (стек, границы пакетов, интеграции). Новые записи добавляют в эту папку с именем `ADR-NNNN-short-title.md` (последовательный номер).

## Индекс

| ADR | Title |
|-----|--------|
| [ADR-0001](./ADR-0001-monorepo-tooling.md) | Monorepo: pnpm, Turborepo, границы пакетов |

## Как добавить ADR

1. Скопируйте [ADR-TEMPLATE.md](./ADR-TEMPLATE.md).
2. Заполните Context, Decision, Consequences, Alternatives.
3. Добавьте строку в таблицу «Индекс» выше.
